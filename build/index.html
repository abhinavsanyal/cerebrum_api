API Endpoints

The API has the following endpoints:

  • GET /users
  • GET /users/{id}
  • POST /users
  • PUT /users/{id}
  • DELETE /users/{id}

API Example

Here is an example of how to use the API to create a new user:

    POST /users
    Content-Type: application/json
    
    {
        "name": "John Smith",
        "email": "john@example.com",
        "password": "password123"
    }
